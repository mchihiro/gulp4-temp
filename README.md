# gulp4-temp

## 開発環境

### インストール

```
$ git clone https://github.com/yoshida-hi/gulp4-temp.git
$ cd gulp4-temp
$ yarn
```

### node-version

```
v10.15.1
```

### Development

```
$ yarn dev
```
（Clean & Build & watch）

### Build

```
$ yarn build
```
（html & js & css & img）

### Clean

```
$ yarn clean
```
